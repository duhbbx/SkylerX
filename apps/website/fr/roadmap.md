# Feuille de route

> Dernière mise à jour : 2026-05-31 · Plan directionnel, sans engagement ferme. La cadence réelle dépend des retours et des ressources.
> Source complète : [ROADMAP.md sur GitHub](https://github.com/duhbbx/SkylerX/blob/main/ROADMAP.md)

## Légende

- ✅ Livré · 🟢 En cours / ce trimestre · 🔵 Prochain trimestre · ⚪ Candidat · 🟣 Long terme

## 1. Bases de données

### Déjà prises en charge

MySQL · MariaDB · PostgreSQL · SQLite · H2 · Oracle · SQL Server · DM (达梦) · KingbaseES · OceanBase · TiDB · GBase · ClickHouse · Snowflake · Redshift · Apache Doris · StarRocks · DuckDB · TDengine · MongoDB · Redis · Elasticsearch

### 🟢 2026 T3

- **PolarDB-PG / -X** (cloud)
- **GaussDB** (Huawei, mode PG)
- **TimescaleDB** (séries temporelles)
- **Cassandra / ScyllaDB** (CQL)
- **InfluxDB 3.x** (FlightSQL)

### 🔵 2026 T4

- **Trino / Presto** (requêtes fédérées)
- **Apache Hive** (HS2)
- **Neo4j** (graphe, Cypher)
- **Couchbase** (N1QL)
- **AWS DynamoDB** (PartiQL)
- **pgvector / Milvus / Qdrant** (vectoriel)

### ⚪ 2027 S1

Apache IoTDB · Nebula Graph · SequoiaDB · GreatSQL · Hologres · Lindorm · TDSQL-C · QuestDB · Druid · Pinot · Flink SQL Gateway · Materialize · RisingWave · Vertica · BigQuery · Athena

## 2. Fonctionnalités

| Catégorie | Prochaines étapes |
|---|---|
| **Éditeur** | Mode Notebook · Visual Query Builder · Speech-to-SQL · Traducteur de SP entre dialectes |
| **Grille** | Vue Form · Filtre multi-valeurs (style Excel) · Master/Detail · Lookup FK |
| **Schéma** | Diagramme ER auto-disposé · Ingénierie directe · Migration v2 |
| **DBA** | Détection d'index morts · Réécriture IA des requêtes lentes · Dashboard de réplication |
| **IA** | Mock data v2 (FK-aware) · Health check v2 · Complétion streaming · RAG sur schéma |
| **Collab.** | Synchronisation de connexions E2E · Bibliothèque d'équipe · Édition web |
| **Export** | Visualiseur de graphiques (ECharts) · Push vers BI (Metabase / Superset / PowerBI / Tableau) |
| **Plateforme** | Signature de code (macOS / Windows) · Sentry · E2E Playwright |
